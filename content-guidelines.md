# Images
Every character has 3 images

## Big image
500x500 png image with transparent background. Used in battles and dialogs

## Horisontal image
80x40 png image with opaque background. Used for actions timeline. Usually created by cropping the big image

## Fallback image
50x70 png image with opaquie backround. Used for character portraits in various menus. Usually created by cropping the big image
